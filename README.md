# ⛽ Fuel Price Radar

Fuel Price Radar is a web application that displays fuel prices using the **Tankerkoenig API**.

The application retrieves real-time fuel price data and displays nearby gas stations.

This project demonstrates how to combine a **Node.js backend** with a **JavaScript frontend**.

---

## 🚀 Features

- Fetch fuel prices from the Tankerkoenig API
- Display nearby gas stations
- Backend server using Express
- Simple and clean user interface

---

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express
- Tankerkoenig API
